# BigFiveBuys
A secure South African C2C e-commerce platform built using HTML, CSS, JS, PHP, and MySQL.

## 🔧 Tech Stack

- **Frontend:** HTML, CSS, JavaScript (jQuery)
- **Backend:** PHP
- **Database:** MySQL
- **Optional Framework:** Bootstrap

---

## ✅ Features

- 🔐 User registration & login with role-based access (admin/customer)
- 🛒 Product listing and browsing interface
- 🧑‍💼 Admin panel for managing users and products
- 📊 Report generation and analytics (dummy data included)
- 💬 Secure (simulated) communication between buyers and sellers
- 📱 Responsive UI/UX for mobile and desktop devices
- 🔎 Search and filter capabilities for faster browsing

## 🎯 Project Goal

This project aims to **empower informal businesses and individual sellers in South Africa** by providing a **trusted, secure, and accessible digital marketplace**. It tackles problems like:

- Fraud and lack of identity verification
- Poor user feedback systems
- Limited trust in online payments
- Lack of local ownership and support

With over **70% of South African consumers citing security concerns in e-commerce**, this platform is built to **restore trust and drive local digital trade**.
